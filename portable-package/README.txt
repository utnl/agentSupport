🍌 BANANA CAT ASSISTANT - PORTABLE VERSION
==========================================

📦 CÁCH SỬ DỤNG:
1. Giải nén toàn bộ thư mục này
2. Double-click "assistant-tdc.exe"
3. App sẽ chạy và ẩn vào system tray (góc dưới bên phải)
4. Click icon 🍌 trong tray để mở app

- Sau đó có thể dùng offline
